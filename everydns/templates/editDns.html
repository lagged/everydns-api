Mass IP Renumbering

When you move servers, renumbering can be very painful. This tool allows the system to look for a given IP/IP Stem and replaces it with a new IP/IP stem.

WARNING! EveryDNS does not currently support editing. This means that this tool is required to delete the old record and add the new record. There is a possibility that if something goes wrong you could lose data. Please make sure and print out the current DNS settings before uisng this tool.

192.168.0.
